https://typoraio.cn/

https://www.bilibili.com/video/BV1iHaTe9EZ3

https://pan.baidu.com/s/17CunIoob5JbtQZ1N5ZIyfg?pwd=zspz

将破解软件复制到Typora安装目录下后，管理员方式打开cmd，进入Typora安装目录，执行下列命令得到激活码

```bash
.\node_inject.exe
.\license-gen.exe
```

## 11.git

https://git-scm.com/

若重装系统后使用原仓库，git会检查到文件所有者和使用者不一致，拒绝操作，此时只需执行下面的代码，把当前目录设为安全，即可继续使用  

```bash
git config --global --add safe.directory D:/typora/Study-Notes
```

（把后面的目录改成仓库目录即可）
